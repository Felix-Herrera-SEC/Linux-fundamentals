
# Linux Navigation

## Objectives

Find a .txt file called text.txt thats hidden inside layers of the non hiden directory using ls command to see whats inside the directories and cd command to access the directory, then use the cat command to open the .txt file  and see what message is inside, also use the pwd command to see the full file path to the .txt file. addionatilly use ls -l to see all the directory in detail and the ls -la command to see the hidden directory as well.

-Ubuntu VM

-Ubuntu

## steps performed

1- went to ubuntu.com to download .iso file of Ubuntu 26.04 LTS

2- Completed installation of Ubuntu inside a VM powered by virtual box

4- ran /root command inside terminal to familirize myself with prompt

5- ran basic commands inside the terminal like pwd , ls and Cd.
